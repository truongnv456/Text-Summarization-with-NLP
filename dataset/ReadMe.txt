This is the datasets folder containing three datasets.
Datasets_1: For Text Summarization
  +) test_tokenized   : Dataset for testing process
  +) train_tokenized  : Dataset for training process
  +) val_tokenized    : Dataset for validating process
Datasets_2: For Task Summarization Extraction
Datasets_3: For Named Entity Extraction for Tasks
  +) test_tokenized   : Dataset for testing process
  +) train_tokenized  : Dataset for training process
  +) val_tokenized    : Dataset for validating process
